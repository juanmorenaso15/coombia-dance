# Colombia Dance

Aplicación móvil para explorar y aprender sobre las danzas tradicionales de Colombia.

---

## Definición del Problema

Colombia es un país con una riqueza cultural invaluable, especialmente en lo que respecta a sus danzas tradicionales. Cada región del país tiene expresiones artísticas únicas que representan la identidad, historia y mestizaje de nuestro pueblo.

Sin embargo, **existe una problemática**: muchas personas, especialmente las nuevas generaciones, desconocen la diversidad de danzas colombianas. No saben cuáles son, de dónde vienen, qué instrumentos las acompañan o cómo se bailan. Esta falta de conocimiento pone en riesgo la preservación de nuestro patrimonio cultural inmaterial.

**Colombia Dance** nace para resolver esta problemática, ofreciendo una plataforma digital interactiva que permite:

- **Descubrir** danzas tradicionales de todas las regiones
- **Filtrar** por región geográfica
- **Guardar** danzas favoritas
- **Conocer** la historia y características de cada danza

---

## Justificación del Proyecto

La aplicación contribuye a:

1. **Preservación cultural**: Digitaliza y difunde el conocimiento sobre nuestras tradiciones.
2. **Educación**: Facilita el aprendizaje sobre el folclor colombiano.
3. **Accesibilidad**: Pone la información al alcance de cualquier persona con un dispositivo móvil.
4. **Identidad nacional**: Fortalece el sentido de pertenencia y orgullo por nuestra cultura.

---

## Funcionalidades

- Listado completo de danzas colombianas
- Búsqueda por nombre de danza
- Filtro por región (Caribe, Andina, Pacífica, Llanos Orientales)
- Sistema de favoritos (marcar danzas preferidas)
- Contador de danzas favoritas
- Información detallada de cada danza
- Interfaz moderna e intuitiva

---

## 🛠 Tecnologías Utilizadas

- **React Native** - Framework para desarrollo móvil
- **Expo** - Plataforma para desarrollo y despliegue
- **React Hooks** - Manejo de estado (useState)
- **StyleSheet** - Estilos nativos de React Native
- **FlatList** - Renderizado eficiente de listas
